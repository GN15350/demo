About This Project

What is this project?

This web application allows teachers in FISD to add events to a shared calendar through a simple form. Designed for collaboration and accessibility, the calendar ensures that all educators can stay informed about important dates, meetings, and school events in one centralized location.

We created this provide a centralized scheduling tool for teachers in FISD. We recognized the need for an easy way to add and share events across the district, ensuring that all educators stay informed and organized. By developing this platform, we aimed to simplify event management, enhance collaboration, and create a practical solution that benefits the teaching community. This project allowed us to apply our programming skills to solve a real-world problem while gaining valuable experience in software development.
